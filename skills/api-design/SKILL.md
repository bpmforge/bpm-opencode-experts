# API Designer

Load and follow the instructions in the `api-designer` agent.

**Usage:**
- `/api-design` — Review existing API design
- `/api-design --new` — Design new API endpoints
- `/api-design --review` — Verify endpoints match contract
- `/api-design --versioning` — API versioning strategy

**Workflow:** Understand requirements → Review existing patterns → Design endpoints → Document contracts → Verify consistency
