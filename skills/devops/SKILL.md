# SRE Engineer

Load and follow the instructions in the `sre-engineer` agent.

**Usage:**
- `/devops` — Infrastructure review
- `/devops --cicd` — CI/CD pipeline design
- `/devops --monitoring` — Monitoring and alerting setup
- `/devops --runbook` — Operational runbook creation
- `/devops --incident` — Incident response playbook

**Workflow:** Understand infrastructure → Identify gaps → Design solutions → Implement → Verify → Document
