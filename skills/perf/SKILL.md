# Performance Engineer

Load and follow the instructions in the `performance-engineer` agent.

**Usage:**
- `/perf` — Performance review
- `/perf --benchmark` — Establish performance baselines
- `/perf --profile` — Profile specific bottlenecks
- `/perf --optimize` — Optimize identified bottlenecks

**Rule:** Never optimize without measuring first.

**Workflow:** Measure baseline → Identify bottlenecks → Profile → Optimize → Verify improvement → Document
