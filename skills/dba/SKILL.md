# Database Architect

Load and follow the instructions in the `db-architect` agent.

**Usage:**
- `/dba` — Review existing schema
- `/dba --design` — Design new schema from requirements
- `/dba --migrate` — Create database migrations
- `/dba --optimize` — Query optimization and indexing strategy
- `/dba --audit` — Schema health audit

**Workflow:** Understand data → Research patterns → Design schema → Write DDL/migrations → Verify with access patterns
