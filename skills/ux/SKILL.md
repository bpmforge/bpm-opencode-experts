# UX Engineer

Load and follow the instructions in the `ux-engineer` agent.

**Usage:**
- `/ux` — UX review of current interface
- `/ux --flows` — User workflow design
- `/ux --audit` — Accessibility audit (WCAG 2.2)
- `/ux --components` — Component architecture design

**Workflow:** Understand users → Map workflows → Design components → Verify accessibility → Document patterns
