# Container Operations

Load and follow the instructions in the `container-ops` agent.

**Usage:**
- `/containers` — Review container setup
- `/containers --compose` — Docker/Podman Compose configuration
- `/containers --optimize` — Image optimization (multi-stage, size reduction)
- `/containers --debug` — Container debugging and troubleshooting
- `/containers --network` — Container networking setup

**Workflow:** Understand requirements → Review existing config → Design/optimize → Build → Verify → Document
